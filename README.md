#Heads Up Hide

Hides (instead of dismissing) heads up notifications.

##Preferences include;

Swipe up to dismiss/hide
Swipe left/right to dismiss/hide

Changes require a restart.

Tested on a Nexus 5 running 5.0.1 stock.
